                          # 简介

                          向量是既有方向又有大小的量，在物理当中称为矢量.与之相对的是标量.

                          # 表示方法

                          在印刷体中一般使用小写字母之后加粗进行表示 v，手写则需要加上箭头 v→.

                          # 自由向量

                          只研究向量的大小和方向，不研究起点和终点，一般而言会将原点设置为起点.与之相对的是固定向量.

                          # 加减

                          只有同型向量可以进行加减，当一个二维向量相加，其中分量进行相加即可.

                          设

                          v=[21]w=[32]

                          则 v+w

                          v+w=[2+31+2]=[53]

                          减法和其类似，将正号改为负号即可.

                          # 数乘

                          向量和数进行相乘，向量中的每一个分量进行相加即可.

                          2v=2[21]=[21]+[21]=[2×21×2]=[42]
