## 简介

### Linux 是什么

Linux，严格来说指的是 Linux 内核，它是一个开源的操作系统内核。日常所说的 Linux 通常指代的是基于 Linux 内核的操作系统，这些系统包含了内核、GNU 工具集以及其他各种软件，共同构成了一个完整的操作系统环境。 它以其稳定性、安全性、灵活性以及强大的开源社区支持而闻名，广泛应用于服务器、嵌入式系统、超级计算机以及个人桌面等领域。

### Linux 的起源和发展

### Linux 主要特点

- **开源:** Linux 内核及大部分相关软件的源代码都是公开的，任何人都可以查看、修改和分发。
- **稳定:** Linux 内核经过了大量的测试和优化，具有很高的稳定性和可靠性。
- **安全:** Linux 系统的安全性设计良好，并且开源的特性使得安全漏洞更容易被发现和修复。
- **灵活:** Linux 可以运行在各种不同的硬件平台上，并且可以根据用户的需求进行定制。
- **强大的社区支持:** 拥有一个庞大而活跃的社区，可以提供丰富的文档、教程和技术支持。

## 分类

Linux 发行版种类繁多，它们基于相同的 Linux 内核，但包含不同的软件包、桌面环境和系统工具，以满足不同的用户需求。

### 用途

- **服务器发行版:** 例如 CentOS, Debian, Ubuntu Server, RHEL（Red Hat Enterprise Linux）。 这些发行版注重稳定性、安全性以及服务器相关的功能，常用于构建 Web 服务器、数据库服务器、邮件服务器等。
- **桌面发行版:** 例如 Ubuntu, Fedora, Linux Mint, Manjaro, elementary OS。 这些发行版注重用户友好性和桌面环境的易用性，适合普通用户日常使用。
- **嵌入式发行版:** 例如 OpenWrt, Yocto Project。 这些发行版针对嵌入式设备进行优化，体积小巧，资源占用低。
- **专门用途发行版:** 例如 Kali Linux (用于渗透测试), Scientific Linux (用于科学计算)。

### 包管理器

Linux 发行版使用不同的包管理系统来安装、更新和卸载软件。

- **DEB:** 例如 Debian, Ubuntu, Linux Mint。 使用 .deb 格式的软件包。
- **RPM:** 例如 Fedora, CentOS, RHEL。 使用 .rpm 格式的软件包。
- **Pacman:** 例如 Arch Linux, Manjaro。 使用 .pkg.tar.zst 格式的软件包。

### 基于起源

很多 Linux 发行版之间存在渊源关系。

- **Debian 家族:** 包括 Debian, Ubuntu, Linux Mint 等。
- **Red Hat 家族:** 包括 Fedora, CentOS, RHEL 等。
- **Arch Linux 家族:** 包括 Arch Linux, Manjaro 等。

## 安装

你可以选择一个合适的发行版进行安装，推荐使用 Ubuntu，该发行版用户友好，易于上手。

你也可以采用 WSL 在 Windows 中进行开发。
