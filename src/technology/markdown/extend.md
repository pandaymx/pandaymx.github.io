## 表格

### 基本语法

Markdown 表格由管道符号 `|` 和破折号 `-` 组成，用于分隔单元格和创建表头与表体。

```md
| Header 1 | Header 2 |
|----------|----------|
| Row 1    | Data 1   |
| Row 2    | Data 2   |
```

### 对齐方式

你可以通过在破折号两侧添加冒号 `:` 来设置单元格内容的对齐方式。

- **左对齐**：默认对齐方式。
- **右对齐**：在破折号的右边添加冒号。
- **居中对齐**：在破折号的两边都添加冒号。

```md
| 左对齐 | 居中对齐 | 右对齐 |
|:------ |:-------:|------:|
| 左边    | 中间    | 右边  |
| 左边    | 中间    | 右边  |
```

### 表格内容

表格单元格可以包含文本、链接、图片、代码块等 Markdown 元素。

```md
| 标题      | 链接                   | 代码块     |
|----------|------------------------|-----------|
| 文本示例  | [点击这里](http://example.com) | `代码示例` |
```

:::tip 复杂语法

更加复杂的语法推荐使用 HTML 来生成表格。

:::

## 脚注

在Markdown中实现脚注，可以使用 [^注释] 语法来定义和引用脚注

```md
这是一个有脚注的文本[^1]。

[^1]: 这是脚注的内容。
```

## 锚点

Markdown 允许你使用锚点来转到 Markdown 的标题处。

```md
[Markdown 表格](#表格)                                        
```

:::tip warn 锚点

所有的标题都是通过 `#标题` 进行跳转的，无论几级标题，如果标题名称相同就需要使用自定义 ID 进行区分。

:::

## 自定义 ID

你可以在标题后面使用 `{#id}` 语法添加自定义 ID

```md
# 目录

- [章节 1](#chapter-1)
- [章节 2](#chapter-2)
- [章节 3](#chapter-3)

## 章节 1 {#chapter-1}

这里是章节 1 的内容。

## 章节 2 {#chapter-2}

这里是章节 2 的内容。

## 章节 3 {#chapter-3}

这里是章节 3 的内容。
```

### 功能与好处

- **导航**：自定义 ID 可以创建锚点，便于导航和跳转。
- **唯一性**：每个 ID 都是唯一的，避免了重复和冲突。
- **增强可读性**：使得文档更加结构化和易于理解。

其 HTML 类似于 `<h2 id="chapter-1">章节 1</h3>`，后期可通过 id，添加自定义 css。

## 定义列表

每个术语后后跟 `:` 空格和具体的定义。

```md
术语 1
:   定义 1

术语 2
:   定义 2
    进一步的解释
:   第二个定义
```

## 删除线

在 Markdown 中，你可以使用 `~~`来创建删除线效果。

```md
这是一个 ~~删除线~~ 示例。
```

## 任务列表

在每一个任务列表项中添加 `-` 和 `[ ]` 方括号。`-` 和 `[ ]` 之间需要有空格，`[x]` 表示被选中。

```md
- [x] 已完成的任务
- [ ] 未完成的任务
- [ ] 另一个未完成的任务
```

## Emoji

Markdown 支持使用 Emoji 来添加图标和表情，使文档更加生动有趣。你可以使用冒号包裹 Emoji 短代码来插入 Emoji。也可以直接粘贴 Emoji 图标。

```md
这是一个笑脸的 Emoji :smile:
这是一个大拇指的 Emoji :thumbsup:
这是一个火焰的 Emoji :fire:
😊
```

## 自动网址链接

在 Markdown 中，URL会自动转化为可点击的链接。你可以直接输入网址，Markdown 会自动识别并创建超链接。

```md
请访问我的网站: https://example.com
```
