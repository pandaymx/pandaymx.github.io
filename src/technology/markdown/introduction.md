## Markdown 是什么

Markdown 是一种轻量级的标记语言，可用于在纯文本文档中添加格式化元素。

使用 Markdown 与使用 Word 类编辑器不同。在 Word 之类的应用程序中，单击按钮以设置单词和短语的格式，并且，更改立即可见。而 Markdown 与此不同，当你创建 Markdown 格式的文件时，可以在文本中添加 Markdown 语法，以指示哪些单词和短语看起来应该有所不同。

## Markdown 优势

- StackOverflow、CSDN、掘金、简书、GitBook、有道云笔记等社区，主流的代码托管平台，如 GitHub、GitLab、BitBucket、Coding、Gitee 等等都支持 Markdwon 写作。
- 绝大部分开源项目的 README、开发文档、帮助文档、Wiki 等都用 Markdown 写作。
- Markdown 是纯文本可移植的，不像 Word 将内容锁定在固有格式和软件中。
- Markdown 是独立于平台的。你可以在运行任何操作系统的任何设备上创建 Markdown 格式的文本。

## Markdown 作用

### 网站

Markdown 是为 web 而设计的，可以轻松地将 Markdown 文本转换为 HTML 页面。VuePress、Hexo、Jekyll、Docusaurus 等静态网站生成器都支持 Markdown。

WordPress 可以通过 Jetpack 插件实现对 Markdown 的支持。

### 文件资料

Markdown 不具备 Word 的所有功能，但基本功能都可以实现，可以使用 Markdown 文档创作工具来创建 Markdown 格式的文档并将其导出为 PDF 或 HTML 格式。

### 笔记

Markdown 可以轻松管理笔记，但是 OneNote 不支持 Markdown 语法。

你可以选择语雀或 Obsidian 等 Markdown 应用程序来创建 Markdown 格式的笔记。

### 文档

Markdown 适合技术文档，GitHub 等代码托管平台都转向 Markdown 创建文档，你可以尝试使用以下工具来生成文档：

- VuePress 是基于 Vue 开发的静态网站生成器
- GitBook 是使用 Git 和 Markdown 来构建书籍的工具，可以输出为 PDF，ePub，Mobi，HTML 等格式

## Markdown 方言

每个 Markdown 应用程序都实现了稍有不同的 Markdown 语法。推荐选择一个比较主流的 Markdown 应用程序，从而保证不会出现兼容性问题。
