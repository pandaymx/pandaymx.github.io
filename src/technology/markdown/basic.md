## 标题

在单词或短语前添加 `#` 创建标题，`#` 的数量表示标题的级别，最多支持六级标题。

```md
# Heading level 1

## Heading level 2

### Heading level 3

#### Heading level 4

##### Heading level 5

###### Heading level 6

Heading level 1
------

Heading level 2
=====
```

:::tip 可选语法

你可以选择在文本下添加任意数量的 `=` 来标识一级标题，或者使用 `-` 标识二级标题，但是不推荐使用这种写法。

:::

:::tip 最佳实践

在 `#` 和标题之间进行分割。

:::

## 段落

使用空白行将文本进行分割即可。

```md
你好

世界
```

:::tip 最佳实践
不要使用首行缩进。
:::

## 换行

在一行的末尾添加两个或多个空格，使用回车键。

```md
你好 
世界
```
:::tip 提示
推荐使用 `<br>` 标签进行换行，你可以然后段落是两个回车，换行是一个回车。 
:::


## 强调

斜体使用一个 `*` 包裹文本，粗体使用两个 `*` 进行包裹，粗斜体使用三个 `*` 包裹。

```md
*斜体*

**粗体**

***粗斜体***
```

## 引用

创建块引用使用 `>` 。

块引用可以包含多个段落，在段落的空白处也加上 `>`。

块引用可以和其他的标签进行嵌套。

```md
> 你好

> 你好
>
> 你好

> 你好
> 
> > 你好
> # 你好
```

:::tip 最佳实践
`>` 建议和文本之间使用空格进行分割。
:::

## 列表

### 有序列表

有序列表通过使用数字 + `.` 进行排序。

```md
1. 你好
2. 你好
```

### 无序列表

无序列表在列表前添加 `-`、`*`、`+` 进行创建。

```md
- 你好
- 你好
```

### 列表嵌套

嵌套其他元素需要缩进四个字符。

```md
- 你好
    1. 你好
    Hello
    > 你是谁
```

### 最佳实践

在有序列表和无序列表中标签和文字之间需要使用空格进行分割。

## 代码

使用 \` 包裹代码即可。

```md
`code`
```

## 分割线

在单独一行中创建多个 `-`。

```md
---
```

:::tip 最佳实践
在分割线的前后添加空白行。
:::

## 链接

链接文本放在中括号内，链接地址放在后面的括号中，链接title可选。如果是网站和 Email 可以直接使用尖括号。

链接中可以直接进行强调。

```md
[超链接显示名](超链接地址 "超链接title")

这是一个链接 [Markdown语法](https://markdown.com.cn "最好的markdown教程")。

<https://www.baidu.com>

**[超链接显示名](超链接地址 "超链接title")**
```

## 图片

图片链接和超链接类似，不同的是相比链接添加了一个 `!`。

```md
![这是图片](/assets/img/philly-magic-garden.jpg "Magic Gardens")
```

## 转义字符

显示原本用于格式化的字符，在字符前添加 `\`。

```md
* \*你好*Hello*
```

如果需要使用 HTML 中的 `<` 和 `&`，你需要使用 HTML 的实义字符进行输出。


## HTML 标签

大部分 HTML 标签都能被正常解析。

```md
<a src="https://www.baidu.com">百度一下，你就知道</a>
```
