## What is Markdown

Markdown is a lightweight markup language that can be used to add formatting elements to plain text documents.

Using Markdown is not the same as using a Word class editor. In an application like Word, click a button to format words and phrases, and the changes are immediately visible. Markdown is different. When you create a Markdown-formatted file, you can add Markdown syntax to the text to indicate which words and phrases should look different.

## Markdown Advantages

- Communities such as StackOverflow, CSDN, Nuggets, Jianshu, GitBook, Youdao Cloud Notes, and mainstream code hosting platforms such as GitHub, GitLab, BitBucket, Coding, Gitee, etc. all support Markdwon writing.
- Most open-source projects such as README, development documentation, help documentation, Wiki, etc. use Markdown writing.
- Markdown is portable text that is not locked in to a proprietary format or software.
- Markdown is independent of the platform. You can create Markdown-formatted text on any operating system and any device.

## Markdown Feature

### Web Site

Markdown is designed for the web and makes it easy to convert Markdown text into HTML pages. Static website generators such as VuePress, Hexo, Jekyll, Docusaurus, etc. all support Markdown.

WordPress can support Markdown through the Jetpack plugin.

### Documentation

Markdown doesn't have all the features of Word, but the basics are available. You can use the Markdown Document Authoring Tool to create Markdown-formatted documents and export them to PDF or HTML.


### Note

Markdown makes it easy to manage notes, but OneNote doesn't support Markdown syntax.

You can choose from Markdown apps like Yuque or Obsidian to create notes in Markdown format.

### Technical Documents

Markdown is suitable for technical documents, and code hosting platforms such as GitHub turn to Markdown to create documents. You can try the following tools to generate documents:

- VuePress is a static website generator based on Vue
- GitBook is a tool that uses Git and Markdown to build books, and can output to PDF, ePub, Mobi, HTML, and other formats

## Markdown Dialect

Each Markdown application implements a slightly different Markdown syntax. It is recommended to choose a more mainstream Markdown application to ensure that there are no compatibility issues.
