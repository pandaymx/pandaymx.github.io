方法

方法的概述

方法（method）是程序中最小的执行单元
类似与c和c++中的函数

方法需要两个步骤

  1. 方法定义
  2. 方法调用

方法的定义和调用

方法的定义

[权限修饰符] [static] [final] 数据类型 方法名(参数列表){
//方法体;
}

static表示是静态方法 可以直接通过方法名来调用方法

其中的参数列表通过

数据类型 变量名;

进行指定，可以指定多个

方法的调用

方法名(参数);

参数表是变量的值

带返回值的方法

方法限定有数据类型

除非定义为void，否则都需要返回一个与方法限定的数据类型相同的值

方法重载

同一个类中方法名相同，但是参数列表不同

当调用方法后会根据其中的数据类型来进行匹配

形参和实参

形参是方法定义中的参数

实参是方法调用的参数

传递基本数据类型是形参的改变不会改变实参

但是引用数据类型传递的是地址值

数组

存储多个相同数据类型的值

数组一旦创建之后，长度不能发生变化

1 数组的定义

数据类型 [] 数组名

虽然也可以类似c和c++中以将[]移到数组名后
但是并不建议

2 静态初始化

数据类型[] 数组名 ={元素1，元素2，元素3，元素4…};

其中的{}显示数组的内容
根据其中的数量确定数组的大小

3 动态初始化

数据类型[] 数组名 = new 数据类型[数组的长度];

4 地址值

打印数组的时候，实际出现的是数组的地址值

不重写对象的toString()方法之前，一般打印的都是对象的地址值

5 数组元素访问

数组名[索引];

索引是标记位置的

从零开始，连续不间断，逐个+1

6 数组的遍历

将数组内容全部取出来

数组的长度:数组名.length

7 数组的初始化值：

整数类型：0

小数类型：0.0

布尔类型：false

字符类型：’\u0000’

引用类型：null

8.数组两种初始化方式的区别

静态初始化：手动指定数组的元素，系统会根据元素的个数，计算出数组的长度

动态初始化：手动指定数组长度，由系统给出默认初始化值

9 二维数组

也是可以使用静态初始化和动态初始化

小驼峰命名法

适用于变量名和方法名

适用于类名

  • 如果是一个单词，那么首字母大写。比如：Demo、Test。

  • 如果是多个单词，那么每一个单词首字母都需要大写。比如：HelloWorld

选择结构

if语句

选择结构是当满足某种条件，就进行某种处理，根据真假来进行选择

if(条件语句1){

​ 语句体1;

}else if(条件语句2){

​ 语句体2;

}……

else{

​ 语句体n+1；

}

if语句中的else if和else都可以不写

具体的流程便是如果某一个if语句的条件语句结果为true，则执行相关的语句体

如果都不满足有else则执行else中的语句

switch语句

switch (表达式) {
case 1:
语句体1;
break;
case 2:
语句体2;
break;
…
default:
语句体n+1;
break;
}

  1. 首先计算出表达式的值
  2. 和case进行比较，一旦有对应的执行相应的语句体
  3. 如果所有case都不匹配则执行default语句

其中的表达式不可以为long类型

循环

  • 初始化语句： 用于表示循环开启时的起始状态，简单说就是循环开始的时候什么样
  • 条件判断语句：用于表示循环反复执行的条件，简单说就是判断循环是否能一直执行下去
  • 循环体语句： 用于表示循环反复执行的内容，简单说就是循环反复执行的事情
  • 条件控制语句：用于表示循环执行中每次变化的内容，简单说就是控制循环是否能执行下去

for循环

for (初始化语句;条件判断语句;条件控制语句) {
循环体语句;
}

①执行初始化语句

②执行条件判断语句，看其结果是true还是false

​ 如果是false，循环结束

​ 如果是true，继续执行

③执行循环体语句

④执行条件控制语句

⑤回到②继续

while循环

初始化语句;
while(条件判断语句){
循环体;
条件控制语句;
}

do while循环

初始化语句;
do{
循环体;
条件控制语句;
}while(条件判断语句);

三种循环的区别

for和while循环，是先判断，再执行。

do…while是先执行，再判断。

当知道循环次数或者循环范围的时候，用for循环。

当不知道循环次数，也不知道循环范围，但是知道循环的结束条件时，用while循环。
