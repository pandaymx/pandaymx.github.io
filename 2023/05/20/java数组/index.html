java数组

发表于 2023-05-20 分类于 java

数组

存储多个相同数据类型的值

数组一旦创建之后，长度不能发生变化

1 数组的定义

数据类型 [] 数组名

虽然也可以类似c和c++中以将[]移到数组名后
但是并不建议

2 静态初始化

数据类型[] 数组名 ={元素1，元素2，元素3，元素4…};

其中的{}显示数组的内容
根据其中的数量确定数组的大小

3 动态初始化

数据类型[] 数组名 = new 数据类型[数组的长度];

4 地址值

打印数组的时候，实际出现的是数组的地址值

不重写对象的toString()方法之前，一般打印的都是对象的地址值

5 数组元素访问

数组名[索引];

索引是标记位置的

从零开始，连续不间断，逐个+1

6 数组的遍历

将数组内容全部取出来

数组的长度:数组名.length

7 数组的初始化值：

整数类型：0

小数类型：0.0

布尔类型：false

字符类型：’\u0000’

引用类型：null

8.数组两种初始化方式的区别

静态初始化：手动指定数组的元素，系统会根据元素的个数，计算出数组的长度

动态初始化：手动指定数组长度，由系统给出默认初始化值

9 二维数组

也是可以使用静态初始化和动态初始化
