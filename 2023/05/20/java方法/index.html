方法

方法的概述

方法（method）是程序中最小的执行单元
类似与c和c++中的函数

方法需要两个步骤

  1. 方法定义
  2. 方法调用

方法的定义和调用

方法的定义

[权限修饰符] [static] [final] 数据类型 方法名(参数列表){
//方法体;
}

static表示是静态方法 可以直接通过方法名来调用方法

其中的参数列表通过

数据类型 变量名;

进行指定，可以指定多个

方法的调用

方法名(参数);

参数表是变量的值

带返回值的方法

方法限定有数据类型

除非定义为void，否则都需要返回一个与方法限定的数据类型相同的值

方法重载

同一个类中方法名相同，但是参数列表不同

当调用方法后会根据其中的数据类型来进行匹配

形参和实参

形参是方法定义中的参数

实参是方法调用的参数

传递基本数据类型是形参的改变不会改变实参

但是引用数据类型传递的是地址值
