                                    # 硬件检测

                                    # cpu

                                    • CPU-Zopen in new window：CPU基本信息，还有其他硬件的基本信息。
                                    • Fritz Chess Benchmarkopen in new window：国际象棋测试。
                                    • Super PIopen in new window：对圆周率进行运算。
                                    • Cinebenchopen in new window：CPU渲染能力。

                                    # 显卡

                                    • 3DMarkopen in new window：最专业的显卡测试，当然也收费。
                                    • GPU-Zopen in new window：显卡基本信息。
                                    • Furmarkopen in new window：俗称甜甜圈，压力测试。
                                    • Frapsopen in new window：可以记录游戏帧数。
                                    • Afterburneropen in new window：微星的显卡超频软件，也可以监控。

                                    # 全

                                    • 鲁大师open in new window：不推荐，因为它的跑分只是简单的测试每一个软件，之后进行相加。
                                    • 游戏加加open in new window：我也不知道为什么这么火。
